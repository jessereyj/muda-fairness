This thesis presents a formal temporal verification of fairness properties in the Multi-Unit Double Auction (MUDA) protocol. We verify three fairness properties over infinite execution traces generated by a deterministic state-transition system (STS), using Linear Temporal Logic (LTL) interpreted over MUDA executions. The MUDA protocol includes buyer bid submission, seller ask submission, multi-unit matching, computation of a uniform clearing price (when defined), and trade execution over seven phases. Each state of the execution includes bids, asks, quantities, and matches; atomic predicates in LTL are interpreted from MUDA states by a fixed interpretation function. The verified properties are Priority Fairness, Quantity Fairness, and Price Fairness (uniform price fairness). The mechanization defines LTL with the temporal operators X (next), F (eventually), and G (always), but the verified properties are invariants defined with G. Infinite traces are generated by iterating the deterministic transition function δ (implemented as step) and using terminal stuttering after the final phase. Operational semantics, execution traces, and the theorems for the verified fairness properties are implemented and machine-checked in the Rocq proof assistant without admitted lemmas. A concrete execution scenario is included to illustrate the execution trace over time and the satisfaction of the fairness properties.
