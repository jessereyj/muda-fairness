CHAPTER 7
CONCLUSION

This thesis studies the formal verification of fairness in the Multi-Unit Double Auction (MUDA) protocol. It asks whether fairness properties can be specified and verified over all executions using Linear Temporal Logic and a deterministic state-transition system. The results show that priority fairness, quantity fairness, and price fairness can be expressed as invariants. These invariants hold for all execution traces generated from initial states.

This thesis makes three main contributions. First, it gives a deterministic state-transition formalization of MUDA, with executions defined by iterating a deterministic transition function. Second, it verifies fairness using LTL in an invariant-based style: each property is formulated as a G-formula (G phi) and shown to hold at every position of every induced trace. Third, it provides a complete mechanization of the model, semantics, interpretation layer, and proofs in Rocq, with no admitted lemmas.

The MUDA protocol is formalized as a deterministic state-transition system. The transition function is delta : State -> State. Finite executions are defined by iterating delta. Infinite traces are obtained by extending terminal states through stuttering. These traces provide the semantic domain for LTL.

Fairness properties are expressed as formulas of the form G phi. Each formula is evaluated over the execution trace. Priority fairness captures the correctness of the deterministic selection step with respect to the priority relations. Quantity fairness captures the conservation of allocated and residual quantities derived from the match record. Price fairness captures the correctness of the clearing price pstar. When pstar is defined, it satisfies the deterministic pricing rule and the marginal bounds.

All three properties are verified for all traces induced by initial states of the form initial_state(bs, as_list). The development includes the transition system, the LTL trace semantics, and the interpretation of atomic predicates. All proofs are complete. No admitted lemmas are used.

This work establishes correctness at the level of the transition system. It does not rely on simulation or selected executions. The properties hold at every position of every execution trace. The framework also makes the connection between operational semantics and temporal logic precise. State-level predicates are interpreted as atomic propositions. These propositions are evaluated along the trace.

The scenario in Chapter 5 illustrates this interpretation. It shows a concrete execution of the protocol. Each step corresponds to a position in the trace. The fairness properties are evaluated at each step. The trace satisfies the invariants.

Several directions follow for future work. Additional properties can be formalized. These include maximality, match finality, and justified rejection. Each can be expressed as a state predicate and lifted to a temporal formula.

Further work can study temporal properties beyond invariants. Examples include eventual termination and phase progression. These properties require temporal operators such as F and X. These operators are already defined in the LTL framework of this thesis.

The model can also be used as a reference specification. It can support validation of implementations of MUDA. System executions can be compared with the verified transition system and its fairness properties.

This thesis provides a formal specification and machine-checked verification of fairness in MUDA. It shows that the fairness properties hold for all executions of the deterministic transition system. It also establishes a precise connection between execution semantics and temporal logic over traces.
